Detailed Description

Set of functions to query information of the ring buffer.

These functions do not affect the data buffer.

void ringbuff_peek ( ringbuff_t *  rb,
uint8_t *  buffer,
uint8_t  peek_size 
)

Read bytes from ring buffer without remove them.

This is useful when you want to check if a sequence of bytes is available in the ring buffer before take the data.

Note that ringbuff_peek doesn't check the buffer size, this means peek_size must fit in the size of passed buffer.

Parameters
[in]rbring buffer to peek
[out]bufferbuffer where data read will be stored
[in]peek_sizenumber of bytes to peek
