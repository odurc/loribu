loribu_t* loribu_create ( uint8_t *  buffer,
uint32_t  buffer_size 
)

Create a ring buffer object.

The ring buffer object can be created from an existing buffer by providing it through buffer parameter. The macro LORIBU_CREATE_BUFFER should be used at buffer parameter when the creation of a new dynamically allocated buffer is required.

Parameters
[in]buffera static or previously allocated buffer or LORIBU_CREATE_BUFFER
[in]buffer_sizesize of the buffer to be created
Returns
pointer to loribu object or NULL if memory allocation fail

uint32_t loribu_read_until ( loribu_t *  rb,
uint8_t *  buffer,
uint32_t  buffer_size,
uint8_t  token 
)

Read bytes from ring buffer until find first occurrence of token.

Before read the ring buffer it'll check if token is available on it. If not, no bytes will be read. A NULL buffer can be used to remove all bytes before token be found. In this case buffer_size is ignored.

Parameters
[in]rbring buffer to read from
[out]bufferbuffer where data read will be stored
[in]buffer_sizeamount of bytes that buffer is able to store
[in]tokena single byte to compare with the read byte
Returns
number of read bytes
